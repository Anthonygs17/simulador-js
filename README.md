# CodingTEC
> Simulador de Inscripción de Cursos de Programación

## Descripción

Este es un proyecto de simulación de inscripción a cursos de programación. Los usuarios pueden explorar diferentes cursos de programación, agregarlos al carrito y completar la inscripción proporcionando su nombre y correo electrónico.

Puedes ver la página en vivo [aquí](https://anthonygs17.github.io/simulador-js/)

## Funcionalidades

- **Mostrar cursos**: El simulador muestra una lista de cursos disponibles para inscripción.
- **Carrito de inscripción**: Los usuarios pueden añadir o quitar cursos del carrito.
- **Confirmación de inscripción**: Al completar la inscripción, se solicita el nombre y correo del usuario y se muestra un mensaje de agradecimiento.
- **Interactividad**: El precio final se actualiza en tiempo real conforme el usuario agrega o elimina cursos del carrito.

## Tecnologías utilizadas

- **HTML**: Estructura básica de la página web.
- **CSS**: Estilos para la presentación de la página y los elementos interactivos.
- **JavaScript**: Lógica de interacción, como agregar elementos al carrito, calcular el precio final y manejar la inscripción.
- **Bootstrap 5**: Framework CSS utilizado para crear una interfaz de usuario moderna y adaptable.

## Uso

1. **Explora los cursos**: Navega por los cursos disponibles.
2. **Agrega al carrito**: Haz clic en los botones de "Inscribirme" para seleccionar los cursos que te interesen.
3. **Confirma la inscripción**: Haz clic en el botón de "Confirmar inscripción" cuando hayas seleccionado todos los cursos.
4. **Proporciona tus datos**: Ingresa tu nombre y correo electrónico para completar la inscripción.
5. **Mensaje de agradecimiento**: Se mostrará un mensaje de agradecimiento y confirmación de que la inscripción fue realizada correctamente.
